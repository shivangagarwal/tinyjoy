# TinyJoy — Game Directory Submissions

**Goal:** Submit tinyjoy.app to browser game directories for SEO backlinks and referral traffic.

**Key challenge:** Most directories want to iframe-embed games directly, not link to an external site. TinyJoy's individual game URLs (e.g. `https://tinyjoy.app/games/color-match`) should be iframe-embeddable and can be submitted as the playable URL. Some directories also accept HTML/JS file uploads.

---

## Standard TinyJoy Description (use for all submissions)

> TinyJoy is a collection of free browser games you can play instantly — no download, no account required. Jump into Color Match, Memory Flip, Number Rush, Pattern Echo, and Word Scramble. Each game takes under 5 minutes, making TinyJoy perfect for a quick break. Web-first, mobile-friendly, and totally free to play. Find your next favorite tiny game at tinyjoy.app.

**Short tagline (for directories with limited chars):**
> 5 free browser games. No download, no account. Play instantly at tinyjoy.app.

| Field | Details |
|---|---|
| URL | https://itch.io |
| Submission URL | https://itch.io/game/new |
| Requires account | Yes — free developer account |
| Accepts external links | **No** — must upload HTML/JS files |
| Status | **Needs board action** |

**Process:**
1. Create a free itch.io account
2. Create a new project → set "Kind of project" to HTML
3. Upload a ZIP file containing `index.html` + all game assets
4. OR create a "collection page" for tinyjoy.app linking to the external site (less ideal)

**What we can do autonomously:** Prepare the game ZIP files for upload, write the project description copy.

**What needs board action:** Creating the itch.io account and uploading files.

**Recommended approach:** Create one TinyJoy collection page on itch.io linking to tinyjoy.app — this gives us a backlink even if we don't host the full game files there.
